# Nexai

Nexai یک وب‌سایت هوش مصنوعی ساخته شده با PHP است که به API جمنای متصل می‌شود. این وب‌سایت با هدف ایجاد یک رزومه قوی و گرافیکی طراحی شده و کاربران را با تجربهٔ کاربری حرفه‌ای و جذاب روبه‌رو می‌کند.  

## ویژگی‌های پروژه

- گرافیک بالا و مدرن: طراحی حرفه‌ای و جذاب با انیمیشن‌ها و ویدیوهای اختصاصی.  
- بک‌گراند اختصاصی با هوش مصنوعی: حتی بک‌گراند سایت با کمک هوش مصنوعی یک ویدیو ساخته شده است.  
- اتصال به API جمنای: کدهای PHP طبق استانداردهای خود زبان به API متصل شده و عملکرد درست دارند.  
- رفع باگ‌ها: تمامی باگ‌های اولیه پروژه با دانش برنامه‌نویسی رفع شده است.  
- کتابخانه‌های پیشرفته: استفاده از کتابخانه‌های قدرتمند برای انیمیشن‌ها و جلوه‌های گرافیکی.  
- آینده‌نگری پروژه: اضافه کردن ساخت و ویرایش تصویر و ویدیو با قدرت هوش مصنوعی در نسخه‌های بعدی.  

## سوالات متداول پروژه

### 1. هدف اصلی پروژه چیست؟
هدف اصلی ساخت Nexai ایجاد یک رزومه قوی و گرافیکی و نمایش توانایی‌های برنامه‌نویسی و طراحی حرفه‌ای است.

### 2. از چه تکنولوژی‌هایی استفاده شده است؟
زبان برنامه‌نویسی PHP و کتابخانه‌های پیشرفته برای گرافیک و انیمیشن.

### 3. نحوه اتصال به API چگونه است؟
کدهای PHP طبق استانداردهای زبان PHP به API متصل شده و داده‌ها را دریافت و پردازش می‌کنند.

### 4. چالش‌ها یا سختی‌های ساخت پروژه چه بوده؟
پروژه دارای باگ‌های زیادی بود که با دانش برنامه‌نویسی همه آن‌ها رفع شد. همچنین طراحی گرافیکی و ساخت انیمیشن‌ها چالش‌برانگیز بود.

### 5. ویژگی‌های گرافیکی یا طراحی سایت چه هستند؟
انیمیشن‌های جذاب، ویدیوهای اختصاصی ساخته‌شده با هوش مصنوعی، و استفاده از کتابخانه‌های پیشرفته برای جلوه‌های بصری.

### 6. آموزش نصب و راه‌اندازی پروژه
برای نصب روی سرور: باید PHP و ماژول‌های مرتبط نصب باشند.  
برای اجرا روی سیستم شخصی: باید محیط PHP و وابستگی‌های پروژه آماده باشد.  
نصب آسان است و پس از آماده‌سازی محیط، پروژه به‌راحتی اجرا می‌شود.

### 7. برنامه‌های آینده یا آپدیت‌های احتمالی پروژه
اضافه کردن ساخت و ویرایش تصویر و ویدیو با قدرت هوش مصنوعی، بهبود عملکرد و امکانات گرافیکی بیشتر و بهینه‌سازی پروژه برای کاربران.
# Nexai

## Project Features

- High and modern graphics: Professional and attractive design with custom animations and videos.
- Custom AI-generated background: Even the website background is created as a video with the help of AI.
- Connection to the Gemini API: PHP codes are connected to the API according to the language's standards and function correctly.
- Bug fixes: All initial project bugs have been resolved with programming knowledge.
- Advanced libraries: Use of powerful libraries for animations and graphic effects.
- Project foresight: Adding image and video creation and editing with AI capabilities in future versions.
